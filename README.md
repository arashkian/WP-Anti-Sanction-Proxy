# WP-Anti-Sanction-Proxy
افزونه رفع تحریم وردپرس — عبور درخواست‌های وردپرس و المنتور از پراکسی برای رفع محدودیت‌های تحریم. تنظیمات پس از ثبت مخفی می‌شوند و به‌صورت رمزگذاری‌شده در پایگاه داده ذخیره می‌گردند.
برای استفاده از افزونه باید پروکسی http تهیه کنید.
آی پی سرور پروکسی و پورت اتصال باید در سرور اصلی وایت لیست شود در غیر اینصورت اتصال برقرار نمی شود.
